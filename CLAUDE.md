# CLAUDE.md — Примерка Колёс

## О проекте

Лендинг для Telegram-бота **Примерка Колёс** — сервиса, позволяющего примерить различные диски на фото автомобиля с помощью ИИ.

Бот: https://t.me/Primerkakoles_bot

## Стек

- Чистый HTML + CSS + JavaScript (без фреймворков, без сборщиков)
- Хостинг: GitHub Pages (файл `index.html` в корне репозитория)
- Язык интерфейса: русский

## Структура проекта

```
primerkakoles/
├── index.html      # Единственная страница (весь CSS и JS встроены)
└── CLAUDE.md       # Этот файл
```

## Структура лендинга

1. **Hero** — ценностное предложение и CTA-кнопка
2. **Как это работает** — 3 шага (загрузить фото → выбрать диски → получить результат)
3. **CTA-блок** — повторная кнопка перехода в бот

## Правила разработки

- Весь CSS и JS встроен прямо в `index.html` (теги `<style>` и `<script>`)
- Никаких npm, никаких сборщиков, никаких CDN-зависимостей — страница работает офлайн
- Весь пользовательский текст на русском языке
- Мобильная адаптация обязательна (mobile-first)
- Поддерживать семантическую разметку HTML5
- Минимальный, чистый дизайн без лишних зависимостей

## Деплой

GitHub Pages автоматически публикует `index.html` из корня ветки `main`.
Коммит в `main` = деплой.
